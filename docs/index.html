Documentation

Here is documentation for some of my projects

Wiretime

Measure the time a packet is on the wire accurately using hardware timestamping. Useful for measuring the impact of traffic congestion and QoS.

Hexend

Send hexdumps copied from Tcpdump/Wireshark. Easily repeat and manually modify useful or problematic frames instead of relying on pcap playback.

Capmon

A Linux capabilities(7) monitor. Monitor capability checks to find what a command requires. Then give yourself those capabilities to run without sudo.
